## WORKFLOW

### Phase 1: CREATE
1. Select template
2. Customize terms
3. Generate draft
4. Validate completeness
5. Submit for review

### Phase 2: REVIEW
1. Route to reviewers
2. Track redlines
3. Negotiate terms
4. Resolve conflicts
5. Finalize version

### Phase 3: EXECUTE
1. Obtain approvals
2. Capture signatures
3. Record effective date
4. Store executed copy
5. Trigger onboarding

### Phase 4: MANAGE
1. Extract obligations
2. Monitor compliance
3. Track deadlines
4. Process renewals
5. Archive expired
