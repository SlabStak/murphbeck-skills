## EMAIL TYPES

| Type | Use Case | Key Focus |
|------|----------|-----------|
| Professional | General business | Clarity, respect |
| Follow-up | After meetings/calls | Recap, next steps |
| Introduction | Self/colleague intro | Value proposition |
| Request | Asking for something | Specific ask, reason |
| Thank-you | Expressing gratitude | Genuine, specific |
| Apology | Apologizing for issues | Accountability, fix |
| Announcement | Sharing news/updates | Key info, impact |
| Decline | Politely declining | Grace, alternatives |
| Reminder | Gentle reminders | Respectful, deadline |
| Feedback | Giving/requesting | Constructive, clear |

## TONE MATRIX

| Audience | Tone | Style |
|----------|------|-------|
| Executive | Concise, results-focused | Bottom-line up front |
| Colleague | Friendly professional | Collaborative |
| Client | Warm, service-oriented | Solution-focused |
| Vendor | Direct, clear | Transactional |
| Cold contact | Respectful, value-first | Brief, intriguing |

## EMAIL STRUCTURE

```
SUBJECT LINE
├── Clear and specific
├── Action-oriented when needed
├── 6-10 words optimal
└── No clickbait

OPENING
├── Appropriate greeting
├── Context setter (1 sentence)
└── Purpose statement

BODY
├── Key points (bullet if 3+)
├── Supporting details
├── Clear ask/action items
└── Deadline if applicable

CLOSING
├── Next steps summary
├── Call to action
├── Professional sign-off
└── Signature
```

- `/email-composer request [ask]` - Request email
- `/email-composer cold [recipient]` - Cold outreach
- `/email-composer rewrite [draft]` - Improve existing email
