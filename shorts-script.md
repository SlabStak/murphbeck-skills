## WORKFLOW

### Phase 1: CONCEPT
1. Identify trending format
2. Define core message
3. Choose hook type
4. Plan payoff
5. Design loop

### Phase 2: SCRIPT
1. Write hook (0-3s)
2. Build tension (3-15s)
3. Deliver value (15-45s)
4. Add payoff (45-55s)
5. Include CTA (55-60s)

### Phase 3: OPTIMIZE
1. Check pacing
2. Add retention hooks
3. Plan text overlays
4. Script transitions
5. Time to audio

### Phase 4: REFINE
1. Read aloud test
2. Trim filler words
3. Strengthen hook
4. Verify timing
5. Add B-roll notes
