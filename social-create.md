## CAPTION FRAMEWORKS

| Framework | Structure | Best For |
|-----------|-----------|----------|
| AIDA | Attention → Interest → Desire → Action | Sales, promo |
| PAS | Problem → Agitate → Solution | Pain points |
| Story | Situation → Complication → Resolution | Connection |
| List | Hook → Points → CTA | Educational |
| Question | Question → Answer → Engagement | Discussion |

## PLATFORM SPECS

| Platform | Characters | Hashtags | Best Practice |
|----------|------------|----------|---------------|
| Instagram | 2,200 (125 visible) | 3-5 | Front-load message |
| TikTok | 4,000 | 3-5 | Conversational tone |
| LinkedIn | 3,000 | 3-5 | Line breaks for readability |
| Twitter/X | 280 | 1-2 | Concise, punchy |
| Facebook | 63,206 | 0-2 | Question-driven |
