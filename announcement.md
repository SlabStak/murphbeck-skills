## WORKFLOW

### Phase 1: DEFINE
1. Identify announcement type
2. Determine audience
3. Set key message
4. Define desired action
5. Choose distribution channel

### Phase 2: DRAFT
1. Write headline
2. Craft lead paragraph
3. Build body content
4. Add supporting details
5. Include CTA

### Phase 3: REVIEW
1. Check stakeholder approval
2. Verify legal/HR sign-off
3. Confirm timing
4. Prepare FAQ
5. Ready distribution

### Phase 4: DISTRIBUTE
1. Send to stakeholders
2. Publish announcement
3. Monitor responses
4. Answer questions
5. Archive communication

## ANNOUNCEMENT STRUCTURE

| Element | Purpose | Position |
|---------|---------|----------|
| Headline | Grab attention | Top |
| Lead | Who/what/when/why | First paragraph |
| Body | Details/context | Middle |
| CTA | Drive action | End |
| Contact | Questions | Footer |
