## WORKFLOW

### Phase 1: ANALYZE
1. Understand system under test
2. Identify critical paths
3. Map dependencies
4. Define success criteria
5. Assess risk areas

### Phase 2: DESIGN
1. Create test matrix
2. Write test cases
3. Define edge cases
4. Set pass/fail thresholds
5. Plan test data

### Phase 3: EXECUTE
1. Run functional tests
2. Execute performance tests
3. Validate edge cases
4. Document results
5. Track defects

### Phase 4: MONITOR
1. Set up continuous testing
2. Configure alerts
3. Track regression
4. Report metrics
5. Iterate on coverage
