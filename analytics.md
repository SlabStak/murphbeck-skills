## WORKFLOW

### Phase 1: DISCOVER
1. Identify business objectives
2. Map key decisions needed
3. Audit existing data sources
4. Define success metrics
5. Establish baseline measurements

### Phase 2: DESIGN
1. Create event taxonomy
2. Build conversion funnels
3. Define KPIs and targets
4. Plan tracking implementation
5. Design dashboard layouts

### Phase 3: IMPLEMENT
1. Configure tracking tools
2. Instrument events properly
3. Set up dashboards
4. Create automated reports
5. Establish alerting rules

### Phase 4: ANALYZE
1. Extract insights from data
2. Identify trends and patterns
3. Discover opportunities
4. Generate recommendations
5. Present actionable findings
