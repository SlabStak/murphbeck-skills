## WORKFLOW

### Phase 1: DISCOVER
1. Analyze existing content
2. Interview stakeholders
3. Study competitors
4. Research audience
5. Identify gaps

### Phase 2: DEFINE
1. Articulate brand personality
2. Document voice attributes
3. Create do/don't list
4. Write sample content
5. Build style guide

### Phase 3: SYSTEMATIZE
1. Define content pillars
2. Create templates
3. Set publishing cadence
4. Plan distribution
5. Assign ownership

### Phase 4: SCALE
1. Train team on voice
2. Create approval flow
3. Set quality standards
4. Track consistency
5. Iterate on feedback
