## WORKFLOW

### Phase 1: PLAN
1. Define research objectives
2. Identify target participants
3. Create screening criteria
4. Select test methodology
5. Build session timeline

### Phase 2: PREPARE
1. Write moderator script
2. Design test tasks
3. Create discussion guide
4. Set up recording tools
5. Schedule participants

### Phase 3: CONDUCT
1. Welcome and brief participant
2. Facilitate think-aloud tasks
3. Observe and document
4. Ask follow-up questions
5. Debrief and thank

### Phase 4: ANALYZE
1. Review session recordings
2. Identify patterns and themes
3. Classify findings by severity
4. Generate recommendations
5. Compile final report

|------|--------------|----------|
| Moderated Remote | 5-8 | Detailed insights |
| Unmoderated | 10-20 | Quick validation |
| In-Person | 5-8 | Complex products |
| Guerrilla | 3-5 | Fast feedback |
| A/B Testing | 100+ | Quantitative data |

| Level | Impact | Action |
|-------|--------|--------|
| Critical | Prevents completion | Fix immediately |
| High | Causes major friction | Fix before launch |
| Medium | Causes minor friction | Fix in next sprint |
| Low | Cosmetic issues | Add to backlog |

| Wrap-up | 5 min | Thanks, incentive |
