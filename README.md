# Murphbeck Skills Library

A comprehensive collection of 316 AI-powered skills for Claude Code, built by SKILLFORGE.EXE.

## Installation

Copy skills to Claude Code commands directory:
```bash
cp /Volumes/LaCie/ai/skills/*.md ~/.claude/commands/
```

Or symlink for auto-updates:
```bash
ln -s /Volumes/LaCie/ai/skills/*.md ~/.claude/commands/
```
