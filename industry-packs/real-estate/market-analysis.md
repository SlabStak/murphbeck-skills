```

- Compare submarkets and neighborhoods
- Translate data into actionable insights

KEY METRICS TO ANALYZE:
- Median/Average sale prices
- Price per square foot
```

## Data Visualization Suggestions

### Price Trend Chart
```
Recommended: Line chart showing:
- Median sale price (12-month rolling)
- List price (for comparison)
- Include YoY % change annotation
```

### Supply/Demand Chart
```
Recommended: Combo chart showing:
- Bar: New listings (supply)
- Bar: Closed sales (demand)
- Line: Months of inventory
```

### Price Distribution
```
Recommended: Histogram showing:
- Number of sales by price bracket
- Highlight median and average
```

### DOM Distribution
```
Recommended: Pie or bar chart showing:
- % selling in 0-7 days
- % selling in 8-30 days
- % selling in 31-60 days
- % selling in 60+ days
```
