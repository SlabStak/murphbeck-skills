## WORKFLOW

### Phase 1: PREPARE
1. Identify ideal customer
2. Research their story
3. Develop question guide
4. Plan production logistics
5. Send prep materials

### Phase 2: INTERVIEW
1. Build rapport
2. Ask discovery questions
3. Probe for specifics
4. Capture emotional moments
5. Get explicit recommendations

### Phase 3: CAPTURE
1. Record main interview
2. Shoot B-roll footage
3. Capture workplace scenes
4. Get product-in-use shots
5. Film additional soundbites

### Phase 4: PRODUCE
1. Select best sound bites
2. Structure narrative arc
3. Add B-roll and graphics
4. Include music and pacing
5. Export final versions

## TESTIMONIAL TYPES

| Type | Duration | Focus |
|------|----------|-------|
| Short | 30-60 sec | Single quote/result |
| Story | 2-3 min | Full journey |
| Results | 60-90 sec | Data-focused |
| Comparison | 90-120 sec | Before/after |
| Compilation | 2-3 min | Multiple customers |
| Case Study | 4-6 min | Deep dive |

## STORY STRUCTURE

| Section | Duration | Purpose |
|---------|----------|---------|
| Introduction | 0:00-0:15 | Who they are |
| The Before | 0:15-0:45 | The problem |
| Discovery | 0:45-1:00 | Finding solution |
| Experience | 1:00-1:30 | Using product |
| Results | 1:30-2:00 | Outcomes |
| Recommendation | 2:00-2:30 | Endorsement |

## QUESTION CATEGORIES

| Category | Purpose | Example |
|----------|---------|---------|
| Background | Establish credibility | "Tell us about your role" |
| Problem | Create relatability | "What challenge did you face?" |
| Discovery | Show research | "How did you find us?" |
| Experience | Build confidence | "What was onboarding like?" |
| Results | Prove value | "What outcomes have you seen?" |
| Recommend | Drive action | "Who would you recommend this to?" |
