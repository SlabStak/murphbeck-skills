## WORKFLOW

### Phase 1: RESEARCH
1. Identify prospect
2. Find trigger event
3. Note specific detail
4. Understand pain point
5. Define value prop

### Phase 2: CRAFT
1. Select framework
2. Write subject line
3. Draft opening hook
4. State value briefly
5. Add soft CTA

### Phase 3: OPTIMIZE
1. Check word count
2. Verify personalization
3. Remove template phrases
4. Test mobile length
5. Remove attachments

### Phase 4: SEQUENCE
1. Plan follow-ups
2. Add new value
3. Schedule sends
4. Track responses
5. Iterate approach
