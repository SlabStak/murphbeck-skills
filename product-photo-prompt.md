## PLATFORM-SPECIFIC TIPS

### Midjourney
```
Add: --ar 1:1 (square), --ar 4:5 (portrait)
Add: --s 250 for commercial style
Add: --style raw for realistic photos
Use: "product photography" as key phrase
```

### DALL-E 3
```
Be specific about lighting and background
Mention "e-commerce" or "commercial" style
Describe exact composition desired
Request "photorealistic" quality
```

### Stable Diffusion
```
Use product photography LoRAs
Add quality boosters in prompt
Use negative prompts for clean results
Consider ControlNet for exact poses
```

## E-COMMERCE REQUIREMENTS

### Amazon/Marketplace Standards
```
Main Image Requirements:
- Pure white background (RGB 255,255,255)
- Product fills 85% of frame
- No text, logos, or watermarks
- High resolution (2000px+)

Prompt Template:
"professional Amazon product photography of [product],
pure white background #FFFFFF, product fills frame,
no text or logos, sharp focus, high resolution,
e-commerce compliant, marketplace ready"
```

### Social Media Optimized
```
Instagram Square (1:1):
"Instagram product photography of [product],
square format, eye-catching, scroll-stopping,
social media optimized, lifestyle aesthetic"

Pinterest Vertical (2:3):
"Pinterest product photography of [product],
vertical format, aspirational, save-worthy,
lifestyle context, inspirational"
```

## COMMON ISSUES & FIXES

| Issue | Cause | Fix |
|-------|-------|-----|
| Unrealistic lighting | Generic prompts | Specify exact lighting setup |
| Busy background | Too many elements | Add "minimal", "clean", "simple" |
| Wrong scale | No reference | Add size context or props |
| Poor composition | Vague framing | Specify "centered", "rule of thirds" |
| Inconsistent style | Mixed references | Use consistent style keywords |
