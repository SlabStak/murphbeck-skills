## PLATFORM REQUIREMENTS

### ElevenLabs
```
INSTANT VOICE CLONING:
- Minimum: 30 seconds of audio
- Recommended: 1-3 minutes
- Format: MP3, WAV, M4A
- Max file size: 10MB

PROFESSIONAL VOICE CLONING:
- Minimum: 30 minutes of audio
- Recommended: 1-3 hours
- High quality: 44.1kHz, 16-bit minimum
- Multiple recording sessions ideal

QUALITY TIPS:
✓ Clear speech, no background noise
✓ Consistent microphone distance
✓ Natural speaking pace
✓ Varied emotional range
✓ No music or sound effects
```

### PlayHT
```
INSTANT CLONING:
- Minimum: 30 seconds
- Recommended: 2-3 minutes
- Format: MP3, WAV, FLAC

ULTRA-REALISTIC CLONING:
- Minimum: 3-5 minutes
- Recommended: 15-30 minutes
- High quality recordings

QUALITY TIPS:
✓ Studio or quiet room recording
✓ Consistent volume levels
✓ Natural expressiveness
✓ Clean audio, no artifacts
```

### Resemble.AI
```
REQUIREMENTS:
- Minimum: 25 clean sentences
- Recommended: 50-100 sentences
- Format: WAV preferred (44.1kHz/48kHz, 16/24-bit)
- Each clip: 3-15 seconds

QUALITY TIPS:
✓ Record each sentence separately
✓ Natural pauses between sentences
✓ Consistent recording environment
✓ Clear pronunciation
```

## RECORDING ENVIRONMENT

### Ideal Setup
```
ROOM:
- Small to medium room
- Soft furnishings (reduce echo)
- Away from windows/traffic
- No air conditioning noise

TREATMENT:
- Blankets/pillows for absorption
- Closet recording (clothes absorb sound)
- Foam panels if available
- Avoid hard, reflective surfaces
```

### Equipment Recommendations

```
BUDGET ($50-150):
- Blue Yeti USB microphone
- Pop filter
- Boom arm or stand
- Quiet environment

MID-RANGE ($150-500):
- Audio-Technica AT2020
- Focusrite Scarlett Solo interface
- Pop filter + shock mount
- Acoustic treatment

PROFESSIONAL ($500+):
- Shure SM7B or equivalent
- Quality audio interface
- Treated recording space
- Monitoring headphones
```

### Microphone Technique
```
DISTANCE: 6-12 inches from mic
ANGLE: Slightly off-axis to reduce plosives
POSITION: Mouth level with mic center
POP FILTER: 2-3 inches from mic

AVOID:
✗ Speaking directly into mic
✗ Moving during recording
✗ Touching mic or stand
✗ Paper rustling
```

## PRE-RECORDING CHECKLIST

```
ENVIRONMENT
□ Quiet room (no HVAC, traffic, etc.)
□ Soft surfaces to reduce echo
□ Phone on silent
□ "Recording in Progress" sign posted

EQUIPMENT
□ Microphone connected and tested
□ Pop filter in place
□ Headphones for monitoring
□ Recording software ready
□ Correct settings (44.1kHz+, 16-bit+)

VOICE PREPARATION
□ Stay hydrated (water nearby)
□ Warm up voice (humming, scales)
□ Clear throat before starting
□ Avoid dairy/excessive caffeine

RECORDING
□ Test recording (check levels)
□ Gain set appropriately (peaks at -6dB to -12dB)
□ Scripts printed or on silent screen
□ Plan for breaks every 15-20 minutes
```

## QUALITY CHECKLIST

### Good Sample Characteristics
```
✓ Clear speech, easily understood
✓ Consistent volume throughout
✓ No background noise or hum
✓ No mouth clicks or lip smacks
✓ Natural pacing and rhythm
✓ Appropriate emotional range
✓ Clean starts and endings
✓ No audio artifacts or distortion
```

### Common Issues to Avoid
```
✗ Room echo/reverb
✗ Background noise (fans, traffic)
✗ Plosives (hard P and B sounds)
✗ Sibilance (harsh S sounds)
✗ Mouth noise/clicks
✗ Inconsistent volume
✗ Unnatural reading/monotone
✗ Breathing too loud
```

## POST-RECORDING

### Basic Cleanup
```
1. NOISE REDUCTION
   - Remove constant background noise
   - Light touch only (preserve naturalness)

2. NORMALIZATION
   - Even out volume levels
   - Target: -3dB to -6dB peaks

3. TRIMMING
   - Remove long silences
   - Clean up starts and ends
   - Leave 0.5s padding

4. EXPORT
   - Match platform requirements
   - Consistent file naming
   - Backup original files
```
