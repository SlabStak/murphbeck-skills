## WORKFLOW

### Phase 1: PLAN
1. Define newsletter objective
2. Select content type/format
3. Identify target segment
4. Gather source content
5. Set engagement goals

### Phase 2: DRAFT
1. Write compelling subject line
2. Craft preview text hook
3. Structure main content
4. Add supporting sections
5. Include call-to-action

### Phase 3: OPTIMIZE
1. Test subject line variants
2. Optimize for mobile view
3. Check link destinations
4. Verify personalization
5. Review send timing

### Phase 4: DELIVER
1. Final preview check
2. Send test emails
3. Schedule or send
4. Monitor initial metrics
5. Document learnings

|------|-----------|---------|

| Hybrid | Weekly | Original + curated mix |

## SUBJECT LINE FORMULAS

| Type | Formula | Example |
|------|---------|---------|
| Curiosity | [Unexpected] about [topic] | "The lazy way to 10x output" |
| List | [Number] [things] for [outcome] | "7 tools I use daily" |
| Question | [Thought-provoking]? | "Are you making this mistake?" |
| News | [Topic]: [What happened] | "AI update: What it means" |
| Personal | I [did thing], here's what | "I quit meetings for a month" |
| Direct | [Specific benefit/topic] | "How to write faster" |

## QUICK COMMANDS

- `/newsletter create [type] [topic]` - Create newsletter draft

- `/newsletter curated [theme]` - Build curated roundup
- `/newsletter product [updates]` - Product update newsletter
- `/newsletter analyze [metrics]` - Review performance
