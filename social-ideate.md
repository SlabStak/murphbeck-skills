## IDEATION FRAMEWORKS

| Framework | Structure | Use For |
|-----------|-----------|---------|
| 4-H Method | Helpful, Humorous, Heartfelt, Hype | Content variety |
| Trend Remix | Trend + Brand Angle | Viral potential |
| Pain-Solution | Problem → Agitate → Solve | Educational |
| Story Arc | Setup → Conflict → Resolution | Connection |
| List Format | Number + Hook + Value | Easy consumption |
