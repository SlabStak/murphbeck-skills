## WORKFLOW

### Phase 1: MONITOR
1. Scan all channels
2. Collect interactions
3. Detect mentions
4. Track sentiment
5. Flag anomalies

### Phase 2: TRIAGE
1. Classify priority
2. Assign SLA
3. Route to handler
4. Log in queue
5. Set reminders

### Phase 3: RESPOND
1. Assess context
2. Select template
3. Personalize message
4. Send response
5. Log interaction

### Phase 4: REPORT
1. Compile metrics
2. Analyze sentiment
3. Document escalations
4. Identify trends
5. Generate insights

| Priority | Response Time | Examples |
|----------|---------------|----------|
| P1 Critical | 15 min | Crisis, legal, safety |
| P2 Urgent | 1 hour | Complaints, unhappy customers |
| P3 Standard | 4 hours | Questions, feedback |
| P4 Low | 24 hours | General comments, praise |

## RESPONSE TYPES

| Interaction | Response Approach | Template |
|-------------|-------------------|----------|
| Praise | Thank + extend | Gratitude templates |
| Question | Answer + invite | Help templates |
| Complaint | Empathize + resolve | Recovery templates |
| Negative | Acknowledge + offline | De-escalation templates |
| Spam | Hide/delete/block | No response |
