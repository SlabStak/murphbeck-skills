## WORKFLOW

### Phase 1: DISCOVER
1. Identify video type and purpose
2. Define target audience
3. Clarify key messages
4. Establish runtime constraints
5. Gather brand guidelines

### Phase 2: STRUCTURE
1. Craft compelling hook
2. Build problem narrative
3. Introduce solution
4. Stack benefits clearly
5. Design strong CTA

### Phase 3: SCRIPT
1. Write scene-by-scene content
2. Add visual directions
3. Time each section
4. Include voice notes
5. Calculate word count

### Phase 4: POLISH
1. Read aloud for flow
2. Trim excess content
3. Strengthen transitions
4. Verify timing accuracy
5. Finalize production notes

## VIDEO TYPES

| Type | Duration | Purpose |
|------|----------|---------|
| Product | 60-90 sec | SaaS/app explanation |
| Service | 90-120 sec | Service offering |
| Concept | 60-90 sec | Idea/methodology |
| Onboarding | 2-3 min | User training |
| Process | 90-120 sec | How something works |
| Animated | 60-90 sec | Whiteboard/motion |

## SCRIPT STRUCTURE

| Section | Timing | Purpose |
|---------|--------|---------|
| Hook | 0:00-0:10 | Grab attention |
| Problem | 0:10-0:25 | Agitate pain |
| Solution | 0:25-0:45 | Introduce answer |
| Benefits | 0:45-1:10 | Show value |
| Proof | 1:10-1:20 | Build credibility |
| CTA | 1:20-1:30 | Drive action |

## TIMING REFERENCE

| Duration | Word Count | Use Case |
|----------|------------|----------|
| 60 sec | ~150 words | Quick overview |
| 90 sec | ~225 words | Standard explainer |
| 2 min | ~300 words | Detailed explanation |
| 3 min | ~450 words | Training/onboarding |
