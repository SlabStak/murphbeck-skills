## WORKFLOW

### Phase 1: DISCOVER
1. Inventory data assets
2. Classify sensitivity
3. Map data flows
4. Identify owners
5. Document lineage

### Phase 2: GOVERN
1. Define ownership model
2. Establish policies
3. Set quality standards
4. Configure access rules
5. Plan retention

### Phase 3: IMPLEMENT
1. Assign stewards
2. Deploy controls
3. Enable monitoring
4. Train stakeholders
5. Document processes

### Phase 4: MAINTAIN
1. Monitor compliance
2. Audit access
3. Remediate issues
4. Update policies
5. Report status
