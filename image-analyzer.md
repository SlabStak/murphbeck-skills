## ANALYSIS MODES

### General Description
```
Provide a comprehensive description covering:
1. Main subject(s) and their appearance
2. Setting and environment
3. Actions or activities depicted
4. Mood and atmosphere
5. Notable details and context
```

### Accessibility (Alt Text)
```
Create alt text that:
- Describes the image's purpose
- Is concise (under 125 characters ideal)
- Conveys essential information
- Avoids "image of" or "picture of"
- Includes text shown in image
```

### Social Media Caption
```
Generate captions that:
- Hook attention in first line
- Describe relevantly for context
- Include emotional connection
- Suggest relevant hashtags
- Match platform tone
```

### Technical/Product
```
Analyze for:
- Product features visible
- Specifications shown
- Branding elements
- Quality indicators
- Comparison points
```

### Chart/Data Extraction
```
Extract and structure:
- Chart type identification
- Axis labels and values
- Data points/trends
- Key insights
- Tabular representation
```

### Document/OCR
```
Extract all text with:
- Accurate transcription
- Layout preservation
- Language identification
- Formatting notation
- Confidence indicators
```

## DESCRIPTION FRAMEWORKS

### 5W Framework
```
WHO:    People/characters present
WHAT:   Objects, actions, events
WHERE:  Location, setting, environment
WHEN:   Time indicators (if visible)
WHY:    Purpose or context (if inferable)
```

### Spatial Framework
```
FOREGROUND: [immediate elements]
MIDGROUND:  [middle distance]
BACKGROUND: [distant elements]
LEFT SIDE:  [left composition]
RIGHT SIDE: [right composition]
CENTER:     [focal point]
```

### Hierarchy Framework
```
PRIMARY:    Main subject/focus
SECONDARY:  Supporting elements
TERTIARY:   Background/context
DETAILS:    Fine details noticed
```

## STYLE ANALYSIS

### Photography Style
```
Identify:
- Genre (portrait, landscape, product, etc.)
- Lighting (natural, studio, dramatic, etc.)
- Composition (rule of thirds, centered, etc.)
- Color treatment (warm, cool, desaturated, etc.)
- Technical quality (sharp, bokeh, grain, etc.)
```

### Art Style
```
Identify:
- Medium (digital, oil, watercolor, etc.)
- Art movement (impressionism, minimalism, etc.)
- Artist influence (if recognizable)
- Technique observations
- Era/period estimation
```

### Design Style
```
Identify:
- Design category (UI, print, packaging, etc.)
- Style trend (flat, skeuomorphic, brutalist, etc.)
- Typography choices
- Color palette analysis
- Layout approach
```
