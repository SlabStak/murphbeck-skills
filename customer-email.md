## WORKFLOW

### Phase 1: ASSESS
1. Identify issue type
2. Gauge customer emotion
3. Review order history
4. Check policy options
5. Determine urgency level

### Phase 2: ACKNOWLEDGE
1. Validate the concern
2. Show genuine empathy
3. Take ownership
4. Avoid blame language
5. Set resolution expectation

### Phase 3: RESOLVE
1. Present clear solution
2. Explain next steps
3. Provide timeline
4. Offer compensation if needed
5. Include relevant details

### Phase 4: CLOSE
1. Summarize resolution
2. Open door for questions
3. Express appreciation
4. Schedule follow-up
5. Document interaction
