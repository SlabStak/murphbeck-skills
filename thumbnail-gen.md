## WORKFLOW

### Phase 1: ANALYZE
1. Review video content
2. Identify key moments
3. Define target emotion
4. Research competitor thumbnails
5. Select style approach

### Phase 2: DESIGN
1. Choose layout template
2. Plan visual hierarchy
3. Select color palette
4. Draft text elements
5. Position key elements

### Phase 3: COMPOSE
1. Define focal point
2. Apply rule of thirds
3. Balance visual weight
4. Add contrast elements
5. Verify safe zones

### Phase 4: TEST
1. Check mobile preview
2. Verify platform specs
3. Test at small sizes
4. Compare to competitors
5. Plan A/B variants

## QUICK COMMANDS

- `/thumbnail-gen youtube [video]` - YouTube thumbnail design
- `/thumbnail-gen instagram [content]` - Instagram thumbnail
- `/thumbnail-gen tiktok [video]` - TikTok cover image
- `/thumbnail-gen ab-test [current]` - Generate A/B test variants
- `/thumbnail-gen audit [url]` - Audit existing thumbnail
