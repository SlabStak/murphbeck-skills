## SEO ELEMENTS

| Element | Best Practice | Impact |
|---------|---------------|--------|
| Title | 50-60 chars, keyword first | High |
| Meta Description | 150-160 chars, CTA | Medium |
| H1 | Match title, one per page | High |
| H2/H3 | Include LSI keywords | Medium |
| URL | Short, keyword-rich | Medium |
| Alt Text | Descriptive, keywords | Low |
