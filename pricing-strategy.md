## WORKFLOW

### Phase 1: ANALYZE
1. Identify value drivers
2. Segment customers
3. Map competitors
4. Assess willingness-to-pay
5. Define constraints

### Phase 2: DESIGN
1. Structure tiers
2. Allocate features
3. Set price points
4. Define discounts
5. Create bundles

### Phase 3: VALIDATE
1. Test with customers
2. Model scenarios
3. Check margins
4. Verify positioning
5. Refine structure

### Phase 4: GOVERN
1. Document policies
2. Train sales
3. Set review cadence
4. Monitor performance
5. Iterate continuously

## PRICING MODELS

| Model | Best For | Pros | Cons |
|-------|----------|------|------|
| Flat rate | Simple products | Easy to understand | No scaling |
| Per-seat | Collaboration tools | Predictable | Seat sharing |
| Usage-based | Consumption products | Aligned with value | Unpredictable |
| Tiered | Feature differences | Upsell path | Complexity |
| Hybrid | Complex products | Flexibility | Hard to explain |
