## LESSON STRUCTURES

### Traditional (I Do, We Do, You Do)
```
1. HOOK (5 min)
   - Attention grabber
   - Connect to prior knowledge

2. I DO - Direct Instruction (15 min)
   - Teacher models
   - Think-aloud
   - Key concepts explained

3. WE DO - Guided Practice (15 min)
   - Collaborative work
   - Teacher supports
   - Gradual release

4. YOU DO - Independent Practice (15 min)
   - Individual work
   - Apply learning
   - Teacher monitors

5. CLOSURE (5 min)
   - Summary
   - Exit ticket
   - Preview next lesson
```

### Inquiry-Based (5E Model)
```
1. ENGAGE (5 min)
   - Hook curiosity
   - Pose question
   - Activate thinking

2. EXPLORE (15 min)
   - Hands-on investigation
   - Student-led discovery
   - Collect observations

3. EXPLAIN (10 min)
   - Share findings
   - Introduce vocabulary
   - Connect to concepts

4. ELABORATE (15 min)
   - Apply to new context
   - Extend learning
   - Make connections

5. EVALUATE (5 min)
   - Assess understanding
   - Reflect on learning
   - Identify gaps
```

### Project-Based
```
1. LAUNCH (10 min)
   - Introduce driving question
   - Present authentic problem
   - Build interest

2. INVESTIGATE (20 min)
   - Research and explore
   - Gather resources
   - Plan approach

3. CREATE (25 min)
   - Build/design/produce
   - Iterate on work
   - Document process

4. PRESENT (10 min)
   - Share with audience
   - Explain choices
   - Field questions

5. REFLECT (5 min)
   - Self-assessment
   - Peer feedback
   - Next steps
```

### Flipped Classroom
```
BEFORE CLASS:
- Video/reading assignment
- Note-taking guide
- Pre-assessment quiz

IN CLASS:
1. CLARIFY (10 min)
   - Address questions from prep work
   - Clear misconceptions

2. APPLY (30 min)
   - Hands-on activities
   - Problem-solving
   - Collaborative work

3. EXTEND (10 min)
   - Advanced challenges
   - Real-world connections

4. WRAP-UP (5 min)
   - Summary
   - Next prep assignment
```

## DIFFERENTIATION STRATEGIES

### For Struggling Learners
```
- Pre-teach vocabulary
- Provide graphic organizers
- Chunk content into smaller pieces
- Offer sentence stems
- Use manipulatives/visuals
- Allow extended time
- Provide study guides
- Partner with peer tutor
```

### For Advanced Learners
```
- Provide extension activities
- Offer choice boards
- Assign independent research
- Create teaching opportunities
- Add complexity/depth
- Connect across disciplines
- Offer leadership roles
- Provide enrichment materials
```

### For English Language Learners
```
- Use visual supports
- Provide word banks
- Allow native language resources
- Use gestures/realia
- Provide sentence frames
- Pre-teach key vocabulary
- Use graphic organizers
- Partner with bilingual peer
```
