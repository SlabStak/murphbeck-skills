## WORKFLOW

### Phase 1: PREPARE
1. Gather source data
2. Identify key messages
3. Assess audience needs
4. Define decisions needed
5. Set page budget

### Phase 2: STRUCTURE
1. Design report flow
2. Build executive summary
3. Create KPI dashboard
4. Frame decision items
5. Prepare appendix

### Phase 3: DRAFT
1. Write narrative sections
2. Populate dashboards
3. Document recommendations
4. Add supporting data
5. Review for clarity

### Phase 4: DELIVER
1. Pre-read distribution
2. Presentation prep
3. Q&A preparation
4. Meeting facilitation
5. Follow-up actions
