| Attribute | Requirement |
|-----------|-------------|
| Fit | Slim/structured (not loose, not boxy) |
| GSM | 220–260 GSM target |
| Collar | Tight/structured, minimal stretch |
| Branding | Minimal (subtle mark or hem detail) |
| Labels | Woven preferred, clean finishing |
| Shrinkage | Pre-shrunk or controlled spec |
