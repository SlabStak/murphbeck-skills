## WORKFLOW

### Phase 1: DISCOVER
1. Identify value metrics
2. Segment customers
3. Map competitors
4. Gather price signals
5. Form hypotheses

### Phase 2: MODEL
1. Estimate elasticity
2. Build demand curves
3. Simulate scenarios
4. Set guardrails
5. Define targets

### Phase 3: TEST
1. Design experiments
2. Calculate sample size
3. Run A/B tests
4. Collect data
5. Analyze results

### Phase 4: OPTIMIZE
1. Select winning prices
2. Roll out gradually
3. Monitor metrics
4. Adjust as needed
5. Document learnings

## ELASTICITY BENCHMARKS

| Elasticity | Meaning | Pricing Implication |
|------------|---------|---------------------|
| 0 to -0.5 | Inelastic | Room to raise prices |
| -0.5 to -1 | Unit elastic | Optimize carefully |
| -1 to -2 | Elastic | Consider lowering |
| < -2 | Highly elastic | Price very sensitive |
