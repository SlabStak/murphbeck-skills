## WORKFLOW

### Phase 1: INVENTORY
1. List all responsibilities
2. Document key relationships
3. Map access and credentials
4. Identify recurring tasks
5. Note unwritten knowledge

### Phase 2: CAPTURE
1. Write process documentation
2. Record key decisions
3. Document exceptions
4. Create contact lists
5. Export relevant data

### Phase 3: TRANSFER
1. Schedule handoff sessions
2. Walk through documents
3. Shadow live work
4. Answer questions
5. Introduce to stakeholders

### Phase 4: VALIDATE
1. Test knowledge retention
2. Monitor for gaps
3. Establish check-ins
4. Confirm access works
5. Close out transition
